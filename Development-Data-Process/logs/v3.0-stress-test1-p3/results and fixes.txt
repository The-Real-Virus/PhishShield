3. The Errors: Why Chrome Blocked the Screenshot
During your stress test, the service worker threw these errors:

Error: The 'activeTab' permission is not in effect because this extension has not been invoked.

Error: Cannot access contents of the page.

Why this happened:
This is not a bug in your code; it is a strict Chrome security policy. The "activeTab" permission in your manifest.json only allows chrome.tabs.captureVisibleTab() if the user explicitly interacts with the extension (e.g., clicks the popup icon). Because your Python/JS stress test script was opening tabs automatically without user clicks, Chrome blocked the screenshot to prevent background spying.

The Industry Fix:
Because PhishShield is a real-time background scanner, it must be able to take screenshots of dangerous pages automatically without waiting for the user to click the extension icon.

To fix this, we must upgrade the permission from "activeTab" to "tabs".
